The **WHMCS "CNIC" Software Bundle** is a comprehensive solution for managing domains, DNS, and SSL certificates within WHMCS. Developed by **Team Internet Group PLC**, this bundle integrates seamlessly with our Domain Registrar Brand CentralNic Reseller, providing a powerful, all-in-one solution for resellers.

To download the latest release, click [here](https://github.com/centralnicgroup-opensource/rtldev-middleware-whmcs/raw/main/whmcs-cnic-bundle.zip).

---

## Key Features

- **Seamless Integration**: Works with CentralNic Reseller for efficient domain management.
- **Advanced Tools**: Includes modules for domain search, import, migration, monitoring, DNS management, and SSL provisioning.
- **Customizable**: Manage TLD-specific settings and apply DNS templates for consistent setups.
- **User-Friendly**: Designed to simplify workflows and enhance the customer experience.

---

## Quick Start

1. [Download the latest release](https://github.com/centralnicgroup-opensource/rtldev-middleware-whmcs/raw/main/whmcs-cnic-bundle.zip).
2. Unzip the file and upload the contents to your WHMCS `/modules` directory.
3. Follow the [Installation Guide](https://support.centralnicreseller.com/hc/en-gb/sections/13438422570781-WHMCS) for setup instructions.
4. Configure the modules and addons via the WHMCS admin interface.

---

## Included Modules

### 1. **Registrar Modules**
Efficient domain management through CentralNic Reseller with integrated widgets.

### 2. **Domain Addons**
Enhance your domain management with these powerful tools:

| Addon              | Description                                                                                   |
|--------------------|-----------------------------------------------------------------------------------------------|
| **Domain Search**  | Fast, intuitive tool for domain discovery with suggestions and premium support.               |
| **Domain Importer**| Bulk import existing domains from CentralNic brands or other sources.                         |
| **Domain Migrator**| Seamlessly transfer domains between registrars.                                               |
| **Domain Monitoring**| Real-time tracking and alerts for domain status and expirations.                            |
| **DNS Templating** | Apply predefined DNS templates to multiple domains for consistent setup.                      |
| **DNS Manager**    | Full-featured DNS record management for your domains.                                         |
| **SSL Provisioning**| Automated SSL certificate ordering and management.                                           |
| **CNIC Admin Config** ![new](https://img.shields.io/badge/New-blue.svg) | Manage default values for additional TLD fields and advanced registrar settings.            |

> **Note**: The Backorder Addon is currently being revamped and is temporarily excluded from this bundle.
